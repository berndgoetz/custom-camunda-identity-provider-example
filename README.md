# Camunda Custom Identity Service Implementation

This project is based on the implementation from @hashlash
at https://github.com/hashlash/example-camunda-custom-identity-service with a last commit in 2017 :-)

I have completely refactored the example and make it now available as a separate repository, there
where just too many changes to the code. Spring Boot got updated to the latest 3.x version. 
Camunda was updated to the latest version 7 version with Java 25.

# How to Run

Use Java 25, it should also run with Java 21.

Run it as a Spring Boot application, e.g. from your IDE or by using:

```gradlew bootRun```
